Vedi sito

Pubblicare su Netlify

  1. Foto: mettete il JPG nella cartella images/ del progetto (sul computer).
  2. Scheda: compilate il modulo → «Salva progetto».
  3. File: «Scarica file per il sito» → sostituite progetti.json e progetti-data.js nella cartella del sito.
  4. Online: se usate GitHub: commit + push → Netlify aggiorna da solo. Altrimenti inviate i file a chi gestisce il sito.

Il sito è su Netlify (non FTP). Guida completa: file ISTRUZIONI-GESTIONE-PROGETTI.txt
